## 🌟 项目亮点与功能模块 (Features)

* **🎬 动态沉浸式主页**：高清英雄视频/动态背景背景，完美的电极客风格视觉冲击。
* **🏆 英雄与皮肤图鉴展示**：精美排版的英雄卡片，支持流畅的交互动效，多维度展示游戏角色魅力。
* **👥 玩家交流与互动社区**：内置极具游戏氛围的玩家留言板模块，提供模拟社区互动的流畅前端体验。
* **📱 响应式设计 (Responsive Design)**：针对不同屏幕尺寸进行优化，确保在 PC 端和移动端都有极佳的浏览体验。
* **🎨 纯正的本地资产托管**：网站中包含的所有音视频、高清皮肤图片等静态资源均经过本地路径优化，加载迅速，排版整洁。

---

## 🛠️ 技术栈 (Tech Stack)

项目基于经典且现代的前端架构构建，无需复杂的后端配置，纯静态高能交付：

* **HTML5**：构建语义化的网页骨架与多媒体（Video/Audio）原生嵌入。
* **CSS3**：实现炫酷的游戏风 UI、Flexbox/Grid 布局、自定义动画（Animations）以及微交互效果。
* **JavaScript (ES6+)**：驱动页面 DOM 交互、动态数据渲染以及留言板的交互逻辑。
